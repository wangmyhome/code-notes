## 第一章 排序算法

### 选择排序

**思路：**

> 1.首先在未排序序列中找到最小（大）元素，存放到排序序列的起始位置
>
> 2.再从剩余未排序元素中继续寻找最小（大）元素，然后放到已排序序列的末尾。
>
> 3.重复第二步，直到所有元素均排序完毕。

**图解：**

![img](https://www.runoob.com/wp-content/uploads/2019/03/selectionSort.gif)

**代码：**

```java
public static int[] selectSort(int[] a) {
  int n = a.length;
  for (int i = 0; i < n - 1; i++) {
    int min = i;
    for (int j = i + 1; j < n; j++) {
      if(a[min] > a[j]) {
        min = j;
      }
    }
    //交换
    int temp = a[i];
    a[i] = a[min];
    a[min] = temp;
  }
  return a;
}
```
